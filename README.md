## mac_changer

### Getting Started
      ```
      git clone https://github.com/cyberbaaz/mac_changer.git
      cd mac_changer
      python mac_changer.py [options] 
      ```
 Use *-h* for options help 
  
`Python2` and `Python3` both supported
